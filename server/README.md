# Express (server)

micro-project server

## Install the dependencies
```bash
yarn install
```

## Create .env file
```bash
nano .env
```

Inside .env file
```bash
access_token = *****YOUR_PERSONAL_ACCESS_TOKEN*****
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
yarn dev
```

Apps is running on port 3000
